Type 2 Enumeration
The Tappy is capable of heuristically narrowing down the specifications of Type 2 tags it encounters. However, by default, this feature is disabled as it reduces scan performance by increasing the number of operations the Tappy has to perform before reporting the scan. You may enable this feature here if you wish.
Disable Enable
Dual-mode Detection on Write
Detecting Type 1 (Topaz/Jewel) tags requires a different scanning procedure than detecting other types of tags. If you wish to write to type 1 tags, you will need to enable dual-mode detection so that the Tappy knows to switch between the two different scanning procedures when searching for a tag to write to. Enabling this feature does slightly degrade scan performance so it is off by default. Your Tappy has to be running firmware version 0.51 or greater to use this feature.
Disable Enable
